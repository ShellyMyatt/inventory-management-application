# Inventory Management Application

## Overview

This project is a JavaFX-based inventory management system designed to manage products and associated parts within an inventory. It provides a graphical user interface for creating, modifying, and deleting inventory items while ensuring data validation and logical constraints are enforced.

## Features

* Add, modify, and delete parts and products
* Associate parts with products and manage relationships
* Real-time inventory updates through a dynamic UI
* Input validation to ensure data integrity
* Search functionality for parts and products
* Prevents deletion of items that violate business rules

## Technologies Used

* Java
* JavaFX
* Maven
* FXML (UI design)
* Object-Oriented Programming (OOP)

## Key Concepts Demonstrated

* MVC architecture (Model-View-Controller)
* Event-driven programming
* Data validation and error handling
* Separation of concerns using controllers and views
* Use of collections to manage inventory data

## Problem Solved

This application simulates a real-world inventory system where products are composed of individual parts. It demonstrates how to manage relationships between objects, enforce business logic, and provide a user-friendly interface for data manipulation.

## How to Run

1. Open the project in an IDE such as IntelliJ IDEA
2. Ensure Maven dependencies are loaded
3. Run the application from the main JavaFX entry point
